Integration Komito Collector

In order to install Komito Collector on your website simply add a line of code just before closing </body> tag:

<script type="text/javascript" src="https://jscb.googlecode.com/svn/trunk/bin/komito.js"></script>

Default Komito Collector configuration:

<script>
  var _komito = _komito || {
    'trackTwitter': 1,   // Tracks Twitter widgets if they presents on page.
    'trackFacebook': 1,  // Tracks Facebook widgets if they presents on page.
    'trackLinkedIn': 1,  // Tracks LinkedIn plug-ins if they presents on page.
    'trackDownloads': 1, // Tracks download links.
    'trackOutbound': 1,  // Tracks outbound links.
    'trackForms': 1,     // Tracks forms submissions.
    'trackUsers': 1,     // Tracks users logged in to social networks.
    'trackActions': 1,   // Tracks 'mailto', 'tel', 'sms' and 'skype' actions.
    'debugMode': 0       // Enables 'debug' mode.
  };
</script>

Komito Collector configuration parameters:

Social:

Tracks users' interaction with different on page widgets. Statistics available in custom reports or in social report.

trackTwitter — Tries to attach listener to Twitter widget object if it presents on page. Tracking events: favorite, follow, retweet, tweet.

trackFacebook — Tries to attach listener to Facebook widget object if it presents on page. Tracking events: like, unlike, message.

trackLinkedIn — Tries to attach listener to LinkedIn plugin if it presents on page. Tracking events: share, follow.

Downloads:

Tracks downloads by different file types. Statistics available in custom reports or in events report.

trackDownloads — Tracks downloads by different file types.

Default file extensions: mp3, mp4, wma, mov, avi, wmv, mkv, eot, woff, ttf, txt, csv, tsv, pdf, doc, docx, xls, xlsx, ppt, pptx, zip, tar, tgz, bz2, gz, rar, dmg, pkg, 7z, ida, exe, sh, bat

External links

Tracks all outbounding links. Recognises links to social networks and lists those in the social report as well. Statistics available in custom reports or in events report (statistics for links to social networks will be available in social report).

trackOutbound — Tracks all outbounding links.

List of social networks: Google+, Blogger, Facebook, Twitter, LinkedIn, MySpace, VKontakte, Odnoklassniki, Xing, YouTube, Twoo, Reddit, Pinterest, Digg, Foursquare.

Forms

Tracks submitted forms. Statistics available in custom reports or in events report.

trackForms — Attaches event listener to all forms on page.

Logged in Users

Tracks users that are logged in some social network while browsing your site. Statistics available in custom reports or in social report. Networks: Facebook, Google+, Twitter

trackUsers — Tracks users logged in to social networks.

Action links

Tracks action links (mailto, tel, skype, etc). Statistics available in custom reports or in events report.

trackActions — Attaches event listener to action links.

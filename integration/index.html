Integration of Komito Analytics

Adding Komito Analytics to your website is easy – just place this code below, ideally right before the closing </body> tag: 

<script src="//datamart.github.io/Komito/komito.js"></script>

Default Komito Analytics configuration:

  <script>
    var _komito = _komito || {
      'trackTwitter': 1,   // Tracks Twitter events if widget is presented on page.
      'trackFacebook': 1,  // Tracks Facebook events if widget is presented on page.
      'trackLinkedIn': 1,  // Tracks LinkedIn events if plugin is presented on page.
      'trackDownloads': 1, // Tracks files download links.
      'trackOutbound': 1,  // Tracks outbound links.
      'trackForms': 1,     // Tracks forms submissions.
      'trackUsers': 1,     // Tracks pageviews by users logged in to social networks.
      'trackActions': 1,   // Tracks 'mailto', 'tel', 'sms' and 'skype' actions.
      'trackPrint': 1,     // Tracks page print actions.
      'trackMedia': 1,     // Tracks HTML5 video, audio and YouTube players events.
      'trackScroll': 1,    // Tracks scroll depth.
      'debugMode': 0       // Prints all requests to console.
    };
  </script>

Komito configuration parameters:

Social Interaction:

• trackTwitter Tracks Twitter events: favorite, follow, retweet, tweet.
• trackFacebook Tracks Facebook events: like, unlike, message.
• trackLinkedIn Tracks LinkedIn events: share, follow.

Files Downloads:

• trackDownloads Tracks downloads by filename extension.

• Media: mp3, mp4, wma, mov, avi, wmv, mkv
• Fonts: eot, woff, ttf
• Text: txt, csv, tsv
• Docs: pdf, doc, docx, xls, xlsx, ppt, pptx
• Archives: zip, tar, tgz, bz2, gz, rar, dmg, pkg, 7z
• Other: ida, exe, sh, bat

Outbound links

• trackOutbound Tracks all outbound links.

Komito recognizes outbound links to social networks and lists those in the social report as well.
Supported social networks: Google+, Blogger, Facebook, Twitter, LinkedIn, MySpace, VKontakte, Odnoklassniki, Xing, YouTube, Twoo, Reddit, Pinterest, Digg, Foursquare.
Also recognizes Twitter web intents links.

Forms

• trackForms Tracks submitted forms.

Logged in Users

• trackUsers Tracks pageviews by users logged in to social networks.
Supported social networks: Facebook, Google+, Twitter.

Call to action links

• trackActions Tracks CTA links (mailto, tel, sms, skype, etc).

Print

• trackPrint Tracks page print actions.

Media

• trackMedia Tracks HTML5 video, audio and YouTube players events.

Scroll

• trackScroll Tracks scroll depth.

Debugging

• debugMode Prints all requests to console.
Note: For production sites, you can enable debugMode by adding debug=1 query parameter to current page URL.
